# Azure Identity Portfolio

## Portfolio Overview

This repository documents my Azure Hybrid Identity & Cloud Systems Engineering portfolio.

The purpose of this portfolio is to demonstrate hands-on skills in Microsoft Azure, Microsoft Entra ID, hybrid identity, cloud governance, systems engineering, and automation.

This portfolio is being built as a structured learning and documentation project to reinforce cloud skills and demonstrate practical technical ability.

- Azure Systems Engineer
